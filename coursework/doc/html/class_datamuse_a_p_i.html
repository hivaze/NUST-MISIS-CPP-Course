Detailed Description

HTTP API-Клиент Datamuse.

Author
Сергей Братчиков
Date
22/12/2019

Класс представляет из себя C++ API для подбора асоциаций и часто используемых слов к словам/фразам на английском языке

В качестве API используется сайт datamuse.com
